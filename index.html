How It Works

Simple installation, powerful capabilities

01

Install DizziGo

Quick installation with automatic system detection and optimal model selection 

$ curl -fsSL https://get.dizzigo.dev | sh
$ dizzigo --version
02

Configure Your Project

Set up allowed directories and commands for your project 

$ dizzigo init
$ dizzigo config set allowed_dirs ./src ./tests
$ dizzigo config set api_key $OPENAI_API_KEY
03

Start Coding with Dizzi

Launch the AI assistant and start getting intelligent help 

$ dizzigo
